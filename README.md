# Yarn-Docker
Docker files for testing applications that use  Node and Yarn on different operating systems


## Debian

From the root, run one of the following:

### Builder

`docker build Debian/`

## Centos

From the root, run one of the following:

### Builder

`docker build Centos/`

## Ubuntu

From the root, run one of the following:

### Builder

`docker build Ubuntu/`
